## Requirements

- Python 3.12

## Installation

Start by creating a virtual environment (Optional)
```bash
python3 -m venv .venv
source .venv/bin/activate
```

Install the dependencies
```bash
pip install -r pip-dependencies.txt
```
## Running the project

Run the project
```bash
python main.py
```
